Tree

- Traversal types 
    - Level wise 
        - root, node at level 1, node at level 2 ....
    - Pre-order
        - root -> left -> right
    - Post-Order
        - left -> right -> root
    - In Order
        - left -> root -> right 
        - This will start from left-most leaf node and upto root and then to right node. 
        - Results in ordered elements when in-order prints out the elements in sorted order
